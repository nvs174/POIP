== Унарный плюс

Результатом унарного оператора плюса (+) является значение его операнда. Операнд оператора унарного оператора сложения должен иметь арифметический тип. 
Унарный оператор + может изменяет размер переменной

* Пример:
[source, cpp]
----
int main() 
{
 unsigned short x = 5;
printf ("%d\n",sizeof(+x));
printf ("%d\n",sizeof(x));
return 0;
}
----
